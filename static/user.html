📖 使用教程

5 分钟上手，无需任何技术背景

1下载聊天软件

推荐使用 Chatbox — 一款免费的 AI 聊天桌面软件，支持 Windows / Mac / Linux。

下载地址：chatboxai.app
打开后点击"Download"按钮，选择你的操作系统版本下载安装。

也可以用其他 OpenAI 兼容的客户端，如 NextChat、LobeChat 等。本文以 Chatbox 为例。

2注册账号，获取密钥

点击下方按钮进入用户面板：

进入用户面板 →
  1. 在"注册新用户"标签下，输入你的用户名（随便取个名字）
  2. 点击"免费注册"按钮
  3. 系统会自动生成一个 API Key（一串以 sk- 开头的密钥）
  4. 复制保存好这个 Key，下一步要用
⚠️ 如果已经有 Key 了，选"已有 API Key"标签，粘贴查询即可看到余额

3配置 Chatbox

打开安装好的 Chatbox，按以下步骤配置：

  1. 点击设置图标（⚙️ 齿轮按钮）
  2. 在"模型提供方"中选择 OpenAI Compatible
  3. 填入以下三个信息：
API 主机 / 地址：
https://token-proxy-h397.onrender.com
API 密钥 / Key：
sk-xxxxxxxxxxxxxxxxxxxxxxxxxx（你注册得到的 Key）
模型：
deepseek-chat
  1. 保存设置
  2. 回到对话界面，输入消息即可开始聊天

4查余额与充值

在用户面板中，你可以随时查看：

  • 当前余额（元）
  • 账户状态（正常/禁用）
  • 交易记录（每次对话的扣费明细）

快捷查余额：输入你的 Key



余额用完后，联系管理员续费即可。

5常见问题

Q: 我收到"无效的 API Key"怎么办？

检查是否复制了完整的 Key（包括 sk- 前缀），注意不要有多余的空格或换行。

Q: 对话速度怎么样？

取决于网络状况，通常跟用 ChatGPT 差不多，几秒内就有回复。

Q: 可以用手机吗？

Chatbox 有手机端 App（iOS / Android），也可以在手机浏览器中访问网站使用。

Q: 支持其他模型吗？

目前支持多种模型，包括 DeepSeek、智谱 GLM 等。在 Chatbox 模型下拉框中选择即可。
